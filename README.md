# Marstoy Mapper

A browser extension to replace Marstoy product data with more useful data from Rebrickable.

## Install

### Firefox

- Open Firefox and navigate to `about:debugging#/runtime/this-firefox`
- Click `Load Temporary Add-on` and select `manifest.json`

### Chrome-based Browsers

- Open browser and navigate to `chrome://extensions` or `edge://extensions`
- Enable `Developer Mode`
- Click `Load unpacked` and select `manifest.json`

## Usage

- Install the add-on
- Add your Rebrickable API key in the add-on settings
- Visit [Marstoy](https://marstoy.com), the add-on will run automatically as needed
- If it's not working you can enable `Log to Browser Console` and filter for `Marstoy Mapper`

## Notes

You can create a Rebrickable API key at [Rebrickable](https://rebrickable.com/)

This add-on is based on [MarstoyIdConverter-Extension](https://github.com/BjornstadThomas/MarstoyIdConverter-Extension),
it hasn't been updated in a while and is not currently working due to changes on the Marstoy site.

If you run into any problems please create an issue in the repo.
